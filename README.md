# Jetbrains Word2Vec Internship Project

This project was made during interview process for position at Jetbrains Internship Program

## Goals of the project

This project shows how to build skip-gram model with negative sampling without any ML framworks such as
Tensorflow, Keras, etc. and how model works under the hood

## Dataset used

Dataset used in this project is Gutenberg carroll-alice, because words appear in specific contexts, which clearly
shows how Word2Vec builds semantic relationships even in small dataset.
